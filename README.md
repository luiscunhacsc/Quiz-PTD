# Quiz PTD - Modo Desafio para o Teste Escrito

## O que e isto?
Este quiz e um treino rapido para medires como estas **agora** em Psicologia na Transição Digital.

Pensa nele como um check-up:
- mostra o que ja dominas,
- revela o que ainda precisa de reforco,
- ajuda-te a entrar no teste com mais estrategia.

## O que este quiz NAO e
Este quiz **nao substitui** o estudo completo da materia.

E mais para:
- autoavaliar,
- organizar revisao,
- ganhar ritmo.

E menos para:
- ensinar tudo de raiz.

## Como jogar
1. Abre `quiz-ptd.html` no browser.
2. Faz as 10 perguntas com honestidade total:
   - `Sim (Sabia tudo)` = 2 pontos
   - `Em parte (Faltaram detalhes)` = 1 ponto
   - `Nao (Preciso rever)` = 0 pontos
3. No final, ve a tua nota de 0 a 20.
4. Usa a revisao para reforcar os pontos onde ainda ha duvidas.

## Leitura rapida da nota
- **13 valores ou mais**: bom caminho. Agora e consolidar.
- **Menos de 13 valores**: para ja, o foco deve ser mesmo os **outros recursos pedagogicos disponibilizados** de ajuda ao teste escrito.

## Se tiveste menos de 13 valores
Sem drama, isto e informacao util.

Plano de ataque:
1. Vai aos apontamentos, resumos e guias de apoio.
2. Rele os conceitos-chave da unidade.
3. Volta ao quiz para verificar evolucao.

Mensagem importante: este "jogo" serve sobretudo para diagnosticar o teu estado de preparacao.  
Para subir a nota de forma consistente, combina quiz + estudo estruturado dos recursos de apoio.

## Objetivo final
Nao e "sacar uma nota no quiz".  
E chegares ao teste escrito com confianca, clareza e margem para responder bem.
